# Atividade – Programação Web II

## Descrição

Este projeto foi desenvolvido como parte de uma atividade da disciplina **Programação Web II**.
O objetivo foi praticar a criação de páginas web utilizando **HTML & CSS**, aplicando diferentes elementos de estrutura, estilização e interatividade.

## Tecnologias Utilizadas

* HTML5
* CSS3

## Conteúdos Trabalhados

Durante a atividade foram desenvolvidos exercícios envolvendo:

* Listas ordenadas e não ordenadas
* Links para páginas externas
* Tabelas em HTML
* Inserção de imagens
* Criação de formulários
* Estilização com CSS
* Layout com `div`
* Flexbox
* Botões interativos
* Menu de navegação
* Efeitos visuais com `hover` e `transition`
* Modal (janela pop-up)
* Rodapé fixo
